  • Les débuts

  • La rencontre

    Les trois fondateurs se sont rencontrés le 1er avril 2003 dans un magasin de drapeaux, où ils partagèrent leur mécontentement face au monopole présent dans le secteur.

  • La fondation

    Nous nous sommes regroupés dans un garage, nous avons sorti une machine à coudre et nous avons conçu notre premier drapeau le 10 avril 2003.

  • La première vente

    Fiers de notre produit, nous l'avons mis à vendre dans une vente de garage. Les gens se l'arrachaient des mains.


  • L'expansion

  • Ouverture de multiples succursales

    Pour mieux servir nos clients, nous avons ouvert des magasins partout à travers le Québec. Nous en comptons aujourd'hui plus de 25!

  • Acquisition de talents

    Pour faciliter la commande de drapeau, nous avons engagés une équipe de programmeurs et designers hors du commun pour concevoir notre site web.

  • Migration de Subversion vers Git

    Pour que la conception de nos produits soient davantage A G I L E, nous avons migrer notre code vers Git. La technologie est encore nouvelle pour nous, mais notre productivité s'accroit exponentiellement!
